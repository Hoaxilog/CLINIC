APPOINTMENT PROMPTS
Appointment Booking (Not Logged In)

APT-01
Test APT-01 by clicking the Book Appointment button without logging in and submitting valid appointment details. Verify that the appointment is created with a Pending status and a confirmation email is sent.

APT-02
Test APT-02 by submitting an appointment request without logging in and leaving required fields empty. Verify that the system blocks the submission and displays validation errors.
APT-03

Test APT-03 by attempting to access the Appointment History/records history page without logging in. Verify that access is denied and the system prompts the user to log in or register.

Appointment Booking (Logged-in Users)
APT-04
Test APT-04 by logging in with a verified and profile-completed account and submitting an appointment request. Verify that the appointment is created with a Pending status and appears in the user’s appointment history.
APT-05
Test APT-05 by logging in with an unverified account and attempting to book an appointment. Verify that the system blocks the request and prompts email verification.
APT-06
Test APT-06 by logging in with a verified account that has an incomplete profile and attempting to book an appointment. Verify that the user is redirected to the profile completion page.

Staff & Records
APT-07
Test APT-07 by approving or rejecting a pending appointment as a staff or admin user. Verify that the appointment status is updated correctly and an email notification is sent.
APT-08
Test APT-08 by logging in as a registered user and viewing appointment history. Verify that all past and current appointments are displayed.
APT-09
Test APT-09 by logging in as a registered user and accessing treatment records. Verify that only the user’s own treatment records are visible and are read-only.
APT-10
Test APT-10 by booking an appointment while not logged in, then registering an account using the same email address. Verify that previously created appointments are linked to the registered account.
